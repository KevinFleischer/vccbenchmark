Benchmark image files go here.

Name them with an application-set letter followed by an index, matching the
results file and the manifest:

  A1.png A2.png A3.png   B1.png B2.png B3.png   C1.png C2.png C3.png

One set of shots per calendar application. The numbering carries no meaning to
the model; which file is which application/condition is recorded only in the
manifest kept on the reference side.
